## What Kodi Actually Needs

Before diving into specific models, let's understand what Kodi requires from hardware. Unlike Plex servers that need transcoding power, Kodi clients need decoder performance and proper audio handling.

### Video Decoding Requirements

Kodi uses hardware video decoding whenever possible. The key codecs and formats include:

**Essential for 2026:**
- H.264/AVC decoding (universal)
- H.265/HEVC decoding (4K content)
- VP9 decoding (YouTube, some streaming)
- AV1 decoding (newer streaming services)

For 4K HDR content, you need not just decoding but also proper HDR tone mapping. Intel's integrated graphics handle this well. AMD's newer RDNA 2/3 also performs admirably.

**Minimum CPU recommendations:**
- Intel: 8th gen or newer (UHD 600+ graphics)
- AMD: Ryzen 4000 series or newer (Vega 8 or RDNA)

### Audio Passthrough: The Hidden Requirement

If you have a surround sound system, audio passthrough is critical. Kodi needs to pass Dolby TrueHD, DTS-HD Master Audio, and Dolby Atmos bitstreams to your receiver without decoding them first.

**What supports passthrough:**
- Intel HDMI output (excellent support)
- AMD HDMI output (good support on newer APUs)
- USB audio adapters (varies by model)

**What doesn't work well:**
- Some older Realtek audio chips
- Bluetooth audio (latency and bandwidth issues)
- Optical coax (can't carry TrueHD/DTS-HD)

Always test passthrough with your receiver. Some combinations have handshake issues that require firmware updates.

### Remote Control Integration

Kodi without a proper remote is frustrating. The best mini PCs for Kodi support:

**CEC (Consumer Electronics Control):**
- Allows TV remote to control Kodi
- Requires HDMI-CEC adapter (Pulse-Eight)
- Most Intel NUCs and USB HDMI adapters work

**FLIRC USB Receiver:**
- Works with any IR remote
- Maps remote buttons to keyboard commands
- Under $25, universally compatible

**Bluetooth Remotes:**
- Works with built-in Bluetooth on most mini PCs
- Enables voice search on some remotes
- May require specific configuration in Kodi

**Smartphone Apps:**
- Kore (official Kodi remote app)
- Yatse (feature-rich third-party option)
- Requires network connection

## Kodi-Specific Configuration Tips

Hardware is only half the equation. Proper Kodi configuration ensures smooth playback.

### Display and Audio Setup

**Video calibration:**

1. Settings → Player → Videos
2. Enable "Adjust display refresh rate" → "On start/stop"
3. Enable "Sync playback to display"
4. Set "Render method" to "Auto"

**Audio passthrough setup:**

1. Settings → System → Audio
2. Set "Audio output device" to HDMI/DisplayPort
3. Enable "Allow passthrough"
4. Check ALL formats your receiver supports:
   - Dolby Digital (AC3)
   - Dolby Digital Plus (E-AC3)
   - Dolby TrueHD
   - DTS
   - DTS-HD Master Audio
   - DTS-HD High Resolution

**Remote configuration:**

For FLIRC USB:
```ini
# Key mappings for typical Kodi use:
# Direction pad → Arrow keys
# OK → Enter
# Back → Backspace
# Menu → M (context menu)
# Play/Pause → Space
# Stop → X
```

### Optimizing Performance

**Enable hardware decoding:**
1. Settings → Player → Videos
2. "Video decoder" → "Hardware acceleration (auto)"
3. "Render method" → "Auto"

**Reduce skin overhead:**
- Heavy skins like Aeon Nox SiLVO require more GPU
- Estuary (default) runs on anything
- For N100-class devices, stick to lighter skins

**Database optimization:**
```sql
-- If your library is slow, clean the database:
-- Settings → Media → Library → Clean library
-- Then export/import to rebuild
```

**Network optimization:**
For NAS-stored media:
- Use wired Ethernet (1Gbps minimum)
- If WiFi required, use WiFi 6
- Enable "Cache full-sized artwork" locally

## Troubleshooting Common Kodi Mini PC Issues

### Video Stuttering

**Cause:** Typically hardware decoding not working

**Fix:**
1. Verify hardware decode is enabled
2. Check GPU driver is up to date
3. Try different video renderer (Settings → Player → Videos)
4. For Intel: Update to latest Intel graphics driver

### Audio Dropouts

**Cause:** Buffer underruns or passthrough issues

**Fix:**
1. Increase audio buffer (Settings → System → Audio)
2. Try disabling passthrough temporarily
3. Check HDMI cable (use Premium High Speed or Ultra High Speed)
4. Update audio drivers

### Remote Not Working

**Cause:** Driver or configuration issues

**Fix:**
1. For FLIRC: Use FLIRC GUI to reprogram
2. For CEC: Verify adapter connection
3. For Bluetooth: Re-pair device
4. Check Kodi keymap configuration

### 4K HDR Not Displaying Correctly

**Cause:** Display or driver configuration

**Fix:**
1. Verify TV supports HDR (and HDR is enabled in TV settings)
2. Use HDMI 2.0/2.1 port on TV
3. Enable HDR in Windows/OS settings
4. In Kodi: Settings → Player → Videos → "Allow HDR"
